# Rock Paper Scissors

This is a very popular yet simple game called "Rock Paper Scissors". It allows players to choose between rock, paper, and scissors, and determines the winner based on the chosen items.

Getting Started
To get started with the Rock Paper Scissors game, follow these steps:

1. Clone or download the repository.
2. Make sure you have Python installed on your system.

Prerequisites

Python 3.x

How to Play

1. Run the program.
2. Enter your choice by typing 0 for Rock, 1 for Paper, or 2 for Scissors.
3. The program will display your choice and the computer's choice.
4. The winner will be determined based on the game rules:
 #Rock beats Scissors.
 #Paper beats Rock.
 #Scissors beats Paper.
 #If both players choose the same item, it's a draw.

example:

What do you choose? Type 0 for Rock, 1 for Paper or 2 for Scissors. 1
You choose Paper
Computer's Choice is Rock
You Win!

Note

The program uses the random module to generate the computer's choice.
The game will continue to run indefinitely, allowing players to play multiple times.
